About

This extension displays your recent history, recently closed tabs, most visited pages and recent bookmarks in a one click popup menu.

Changelog

1.9.3 - Fixed a few bugs and cleaned up the code a little.

1.9.2 - You can choose what icon you would like in the address bar for Recent History in the options page. Items which have been bookmarked will show a star next them in the history page. The middle-click function has been changed back so when you middle-click an item in the popup window the item opens in a new tab in the background and leaves the popup open.

1.9.1 - Fixed a few bugs with the most visited list. The new history page is the default history manager but this can be changed in options. The new history page shows more search items from the past and not just from your recent history. A few things from the options page have been changed.

1.9 - A brand new history page has been made which can be accessed by clicking on the "Show All History" link. Polish translation has been added, thanks to the person who commented. When you left click an item in the popup menu it opens in the current tab and if you middle click the item, it will open up in a new tab. When the popup or the history page is opened the search bar is focused so you search without clicking on the search field. The recently closed tabs history isn't deleted upon new browser session any more. When you hover over a recent history or most visited item you will be able to see the visit count for that URL. The most visited history items are now gathered from the past 4 weeks which should give you an accurate list of most visited URLs. You can also remove the URLs you do not want to show up in the most visited list in the options page. The popup should open up slightly quicker after you open it the first time. Lastly a few minor changes have been made here and there.

1.8.2 - Fixed issue of the extension causing a high cpu load on some systems like Mac OS and Linux. Please make sure your using the latest stable version of Chrome.

Languages are still supported, however the only way you can use them is to install the locale version of chrome. This is the official way to support localization in chrome extensions. The history manager has been removed for now however I'm working on another new one which will act as your default history page.
